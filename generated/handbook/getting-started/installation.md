# Installation

Dieses Handbuch beschreibt die Installation der **Wissensbasis**.

## Voraussetzungen

- Node.js `>=20`
- Ein Git-Host (GitHub _oder_ Azure DevOps)
- Schreibrechte auf Repo B

## Schritte

1. Repository klonen
2. Abhängigkeiten installieren
3. Konfiguration anlegen

```bash
npm ci
npm run start
```

Weiter mit [Konfiguration](configuration.md).
