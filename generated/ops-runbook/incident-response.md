# Incident Response

Dieser Runbook-Eintrag beschreibt die ersten 15 Minuten.

## Sofortmaßnahmen

1. Alarm bestätigen
2. Blast-Radius eingrenzen
3. Kommunikationskanal öffnen

## Eskalationsstufen

| Stufe | Auslöser    | Reaktionszeit |
| ----- | ----------- | ------------- |
| P1    | Prod down   | 15 Minuten    |
| P2    | Teilausfall | 1 Stunde      |

> Niemals ohne Rollback-Plan deployen.
